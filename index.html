Visualization

This was a project done within excel and tableau. The data was first cleaned within excel, removing duplicates and converting column values to usable values, and then imported to Tableau.
Data from AirBnB in Seattle, Washington, is used to help decide the best area codes to invest in, and how many rooms are needed for the best return on investment. The project discovered the best areas and highest prospects on investment for the client.
